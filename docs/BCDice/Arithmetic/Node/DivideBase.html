Overview

除算ノードの基底クラス

定数 ROUNDING_METHOD で端数処理方法を示す記号( ‘U’, ‘R’, ” ) を定義すること。また、除算および端数処理を行う divide_and_round メソッドを実装すること。

def eval(round_type)
  l = @lhs.eval(round_type)
  r = @rhs.eval(round_type)
  divide_and_round(l, r, round_type)
end
